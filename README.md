This project translates raw operational data into a high-impact, interactive BI dashboard of Maven Market highlighting expertise across the entire Power BI stack:

* Data Transformation: Utilized Power Query for robust ETL processes, including data cleaning, validation, and integration of disparate sources.

* Data Modeling: Implemented an optimized Star Schema data model to ensure query efficiency and reliable reporting relationships.

* Advanced Calculations: Developed complex DAX measures for time intelligence (YTD, YoY), rolling averages, and custom-defined KPIs (Key Performance Indicators).

* Visualization: Designed a professional, intuitive dashboard focused on data storytelling and enabling efficient data-driven decision-making.

**# Key Deliverables**

* Actionable Insights: Clear, segmented views of market performance, profitability, and customer behavior.

* Interactive Reporting: A dynamic dashboard allowing for self-service analysis and drill-down functionality.

* Scalable Architecture: A foundational data model built for future expansion and integration.
